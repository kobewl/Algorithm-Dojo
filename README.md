## 💡 代码模板 | Template

```java
/**
 * 标题：[LC-1] Two Sum - 两数之和
 * 难度：Easy
 *
 * 问题描述：
 * 给定一个整数数组 nums 和一个目标值 target，在该数组中找出和为目标值的两个整数。
 *
 * 示例：
 * Input: nums = [2,7,11,15], target = 9
 * Output: [0,1]
 *
 * 解题思路：
 * 1. 使用 HashMap 存储遍历过的数字和索引
 * 2. 遍历数组，查找 target - nums[i] 是否存在于 HashMap 中
 *
 * @author wangliang
 * @date 2025-02-09
 */
```
